# Analyse Concurrentielle SaaS Devis/Factures Électriciens & Artisans FR

Date : 2026-05-19 • Marché cible : électriciens, artisans BTP, TPE 1-10 personnes • Pour : DEP

---

## 1. Concurrents FR (cœur de cible)

### Tolteck (tolteck.com)
- **Prix** : 25 €HT/mois (sans engagement) ou 19 €HT/mois annuel. Jeune entreprise <5 ans = 149 €HT/an.
- **Top features** : devis/factures, bibliothèque ouvrages + matériaux + main-d'œuvre, fiches clients avec notes, mode hors-ligne chantier, conformité e-facture 2026-2027.
- **UX** : ~4-5 clics pour créer devis depuis bibliothèque. Mobile + desktop + tablette.
- **Spécificité électricien** : aucune réelle (générique BTP), pas de NF C 15-100 native.
- **Relances** : basique, manuelles ou semi-auto.
- **Multi-utilisateurs** : illimités inclus dans le prix (avantage fort).
- **Vocal** : non.
- **Intégrations** : e-facture conforme, pas de Stripe/Pennylane natif.
- **Pain points** : prix au-dessus marché, pas d'export comptable multi-factures PDF, pas de chiffrage avancé, pas multi-devises, abonnement perçu cher en basse activité. Note 4,8 Trustpilot / 4,4 Capterra.

### Sinao (sinao.fr)
- **Prix** : Free / Start 12 €/mois / Standard 25 €/mois / Pro 50 €/mois.
- **Top features** : devis/factures, comptabilité intégrée, suivi dépenses, stock, facturation récurrente.
- **UX** : interface colorée intuitive, accessible non-tech.
- **Spécificité électricien** : aucune (généraliste comptable).
- **Relances** : oui, simples.
- **Pain points** : bugs récurrents (devises factures récurrentes, problèmes envoi email selon avis Trustpilot 4,6/5).

### Henrri (henrri.com, groupe Rivalis)
- **Prix** : version gratuite illimitée (devis/factures), payant pour pré-compta et avancé.
- **Top features** : devis/factures illimités, relances impayés, e-facture via PA Pennylane (gratuit toutes formules).
- **UX** : web simple, courbe d'apprentissage faible.
- **Spécificité électricien** : aucune.
- **Pain points** : pas de facture sans TVA en gratuit (bloque franchise), pas de mobilité avancée, pas de compta intégrée, fonctions avancées payantes mal documentées.

### Obat (obat.fr)
- **Prix** : 25 €HT/mois (annuel) ou 32 €HT/mois sans engagement. 21 000 clients.
- **Top features** : bibliothèque 30 000 ouvrages BTP, déboursé sec, primes énergie/CEE, planning chantier, relances auto email+SMS programmables.
- **UX** : multi-device (PC/tablette/mobile), workflow chantier-centric.
- **Spécificité électricien** : ouvrages BTP couvrent partiellement, pas de NF C 15-100 dédiée.
- **Relances** : **automatiques email+SMS** (force majeure vs concurrents).
- **Multi-utilisateurs** : selon plan.
- **Vocal** : non.

### Mediabat (mediabat.com)
- **Prix** : ~50 €/mois ou licence à vie 956 € (standard).
- **Top features** : chiffrage assisté électricien (neuf + maintenance), suivi chantier, suivi commercial, fonctionnalités très étendues.
- **Spécificité électricien** : module dédié électricité.
- **Pain points** : tarif élevé, courbe d'apprentissage importante, vieille génération UX.

### Codial Bâtiment (codial.fr)
- **Top features** : module Électricien (courant fort/faible, maintenance), CodialPad = devis par glisser-déposer d'icônes/schémas électriques.
- **Spécificité électricien** : **schéma électrique drag-and-drop** (différenciation forte mais niche).
- **Pain points** : interface technique/dense, paramétrage complexe, on-premise heavy.

### Batappli (batappli.fr)
- **Prix** : Devis 39 €HT/mois / Artisan 79 €HT/mois / Entreprise 119 €HT/mois.
- **Top features** : tarifs travaux pré-calculés pour 9 métiers (incl. électricien), génération auto balances + journaux + attestations fiscales, gestion stocks.
- **Cible** : 5+ collaborateurs.

### ProGBat
- **Prix** : à partir de 25 €/mois, essai 14 jours sans CB.
- **Top features** : multi-versions de devis, MAJ prix en 1 clic, tableau de bord chantier.
- **Cible** : TPE 2-5 personnes.

### Vulko (vulko.fr)
- **Top feature unique** : **dictée vocale IA Claude pour devis BTP**. Reconnaissance native navigateur, structuration auto en items BTP, détection quantités/unités, édition avant génération. Devis en 3 min vs 45 min standard.
- **Menace directe** pour DEP (différenciateur vocal déjà occupé). Prix non communiqué publiquement.

## 2. Concurrents internationaux (référence UX)

### Jobber (jobber.com)
- **Prix** : à partir de $29/mois (multi-tiers).
- **Top features** : online booking, scheduling, dispatching, CRM, paiements, app mobile.
- **Pain points** : app mobile lente, features mobile < desktop.

### ServiceM8 (servicem8.com)
- **Prix** : £29–£349/mois par volume jobs.
- **Top features** : digital job cards, photos chantier, formulaires, asset mgmt, **meilleure UX mobile du marché**.
- **Pain points** : iOS-only quasi-exclusif (Android limité).

### Tradify (tradify.com)
- **Prix** : £19–£49/utilisateur/mois.
- **Top features** : iOS + Android équivalents, devis/scheduling/track/invoice mobile-first.

### Fergus (fergus.com)
- **Prix** : $48/utilisateur/mois.
- **Top features** : **conçu pour électriciens + plombiers**, job cards avec photos/notes/checklists, back-costing, site visits, Fergus Pay, relances auto impayés.
- **Référence UX** la plus proche du besoin DEP.

## 4. Top 5 fonctionnalités à VOLER pour DEP

1. **Bibliothèque ouvrages 30k+ items** (Obat) — pré-rempli matériel + main-d'œuvre + déboursé sec, MAJ tarifs fournisseurs en 1 clic.
2. **Relances auto email+SMS programmables** (Obat) — escalation J+7/J+15/J+30, templates personnalisables, anti-impayés.
3. **Job cards mobile avec photos/notes/checklists** (Fergus + ServiceM8) — chaque chantier = dossier vivant terrain, photos avant/après, signature client tablette.
4. **Multi-utilisateurs illimités au même prix** (Tolteck) — abolit la barrière "par siège" qui freine équipe artisan.
5. **Schéma électrique drag-and-drop** (Codial) — visuel artisans non-Excel, mais simplifié vs Codial.

---

## 5. Top 3 différenciants DEP doit avoir

1. **Vocal multilingue auto-corrigé** (vs Vulko mono-FR Claude) — dictée arabe/français/wolof/bambara avec normalisation auto en items électrique standard, détection NF C 15-100 implicite (ex: "tableau 12 modules" → ligne catalogue). Mémoire phrases artisan (apprend dialecte chantier).
2. **Mémoire articles intelligente par chef** — l'app apprend les 50 articles récurrents de CHAQUE chef et les propose en priorité, auto-complète depuis 2 mots vocaux. Aucun concurrent ne fait du ML par utilisateur sur catalogue.
3. **Simplicité radicale chef non-digital** — onboarding ≤2 min sans clavier (premier devis 100% vocal), interface 3 boutons max par écran, mode "WhatsApp" (le devis arrive en message au chef, validé par emoji). Vs Mediabat/Codial/Batappli qui exigent formation.

---

## 6. Anti-patterns à éviter (mauvaises UX observées)

1. **Devis multi-écrans + 15 champs obligatoires** (Mediabat, Codial, Batappli) — décourage chef pressé, perd 70% des artisans <30 ans.
2. **Tarification "par utilisateur" + "par module"** (Fergus $48/user, Batappli 39/79/119) — explose les coûts pour TPE 3-5 personnes, freine l'adoption équipe.
3. **Abonnement sans freemium ni mode "1 devis pour tester"** (Tolteck, Mediabat) — friction inscription trop haute vs Henrri gratuit qui capte 80% des leads marché.

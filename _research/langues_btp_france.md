# Langues parlées sur les chantiers BTP en France

> Rapport de recherche — DEP (plateforme SaaS de traduction patron↔employés)
> Date : 2026-05-19
> Sources : INSEE, DARES, Wikipédia, Cairn, Batiactu, Musée Histoire Immigration, Institut Kurde, persée, openedition.

---

## Contexte chiffré (DARES 2024 + INSEE 2017-2019)

- **Construction = 2e secteur** employant le plus de travailleurs immigrés (19% des emplois où ils sont surreprésentés).
- **Détachement 2024** : 255 900 salariés détachés en France, **40% dans la construction** (≈ 102 000 ETP cumulés sur l'année, 67 100 présents à un instant T en moyenne).
- **Top 5 pays détachement** : Portugal, Roumanie, Pologne, Italie, Espagne. UE = 63% (en baisse).
- **Ouvriers du gros-œuvre** : **27% d'étrangers** (non-qualifiés), 24,8% (qualifiés). Second-œuvre : 14-17%.
- **Maghrébins** : 21,2 % travaillent dans la construction (vs 9,2% pour les Français nés FR).
- **Portugais** : 47,8 % dans la construction. 316 000 immigrés portugais (12% du stock immigré total).
- Île-de-France, PACA, Auvergne-Rhône-Alpes concentrent l'essentiel.

## Top 14 langues classées par fréquence réelle sur chantiers BTP

### 1. 🇫🇷 Français (de référence — non comptabilisé)
Langue cible de traduction. Tous les autres rangs sont relatifs à la **main d'œuvre BTP non/peu francophone**.

### 2. 🇩🇿🇲🇦 Arabe maghrébin (darija) — autonyme : الدارجة
- ISO : `ary` (Moroccan), `arq` (Algerian), `aeb` (Tunisian). Tronc commun mutuellement intelligible.
- **~25-30 %** de la main d'œuvre BTP non-francophone (estimation croisée INSEE).
- Régions : **IDF (93/95/77), Lyon, Marseille, Lille, Nice, Toulouse**.
- Pop. de référence : 5,3 M personnes d'origine maghrébine sur 3 générations (INSEE 2019 + estimation Vermeren). Sur les chantiers : majorité Algériens (gros-œuvre) + Marocains/Tunisiens (gros-œuvre + conducteurs d'engins).
- **À utiliser** : darija (dialectal), PAS arabe standard moderne (MSA). Les ouvriers comprennent mal le MSA littéraire.

### 3. 🇵🇹🇧🇷 Portugais européen — autonyme : Português
- ISO : `pt-PT` (européen) + `pt-BR` (brésilien marginal en BTP).
- **~18-22 %**. 316 000 immigrés portugais en France, 47,8% dans la construction. 38 000 détachés portugais/an (n°2 derrière Pologne).
- Régions : **IDF, Centre-Val-de-Loire, Aquitaine, Pays-de-la-Loire**.
- **Différencier** : `pt-PT` impératif (vocabulaire chantier ≠ BR). Le BR concerne <2% des Lusophones BTP en France.
- Source : Cairn, openedition.

### 4. 🇵🇱 Polonais — autonyme : Polski
- ISO : `pl`.
- **~12-15 %**. **38 000 salariés détachés polonais en France (n°1)**. Surtout TP, gros-œuvre, isolation, charpente.
- Régions : **IDF, Grand-Est, Hauts-de-France, Auvergne-Rhône-Alpes**.
- Source : DARES 2024.

### 5. 🇷🇴 Roumain — autonyme : Română
- ISO : `ro`.
- **~10-13 %**. 27 000 détachés roumains/an (n°3). Forte croissance post-2007 (UE).
- Régions : **IDF, PACA, Occitanie, Grand-Est**. Bcp d'intérim et sous-traitance second-œuvre.
- Source : DARES.

### 6. 🟦 Berbère / Tamazight (kabyle, chleuh, rifain)
- Autonymes : **Taqbaylit** (kabyle), **Tachelhit** (chleuh), **Tarifit** (rifain).
- ISO : `kab` (kabyle), `shi` (chleuh), `rif` (rifain).
- **~6-9 %** de la main d'œuvre BTP (sous-ensemble de la pop. maghrébine, NON couvert par la darija).
- Régions : IDF, Lyon (kabyle très présent), Marseille (chleuh marocain).
- **Critique** : ~1/3 des Algériens en France parlent kabyle en L1, beaucoup ne maîtrisent ni MSA ni darija standard. Idem chleuh chez Marocains du Souss.
- Source : Wikipedia Arabe maghrébin, Yves Montenay.

### 7. 🇹🇷 Turc — autonyme : Türkçe
- ISO : `tr`.
- **~4-6 %**. Accord bilatéral 1965, vague historique IDF, Alsace, Lorraine, Rhône-Alpes. Très présent **second-œuvre (peinture, plâtre, plomberie)**.
- Source : Wikipédia Diaspora turque en France.

### 8. 🇲🇱 Bambara / Mandingue — autonyme : Bamanankan
- ISO : `bm`. Inter-compréhension forte avec malinké (`man`), dioula (`dyu`).
- **~3-5 %**. Diaspora malienne très liée BTP IDF (foyers Sonacotra/Adoma → gros-œuvre, démolition, manœuvres).
- Régions : **IDF (Montreuil, Bagnolet, La Courneuve)**, Normandie.
- Source : persée, openedition INED.

### 9. 🇸🇳 Soninké — autonyme : Sooninkanxanne
- ISO : `snk`.
- **~3-4 %**. Dans les années 1970, **~70% des Subsahariens immigrés en France étaient soninkés**. Aujourd'hui encore très ancrés BTP IDF (Mantes, Pantin, 93).
- Diaspora multi-pays : Mali / Sénégal / Mauritanie. Communauté très solidaire sur chantiers.
- Source : Wikipédia Soninkés, INED.

### 10. 🇸🇳 Wolof — autonyme : Wolof
- ISO : `wo`.
- **~2-4 %**. Diaspora sénégalaise (mais aussi Gambie). Plus présente commerce/services que BTP, mais en croissance second-œuvre depuis 2010.
- Source : Wikipédia diaspora sénégalaise, openedition REMI.

### 11. 🟧 Peul / Pular / Fulfulde — autonyme : Fulfulde / Pular
- ISO : `ff` (macrolangue), `fuc` (Pulaar Sénégal/Mauritanie), `fuf` (Pular Guinée).
- **~2-3 %**. Communautés guinéennes (forte immigration récente 2015+) et mauritaniennes BTP IDF.
- Régions : IDF, banlieue Marseille.

### 12. 🟧 Kurde (Kurmandji) — autonyme : Kurmancî
- ISO : `kmr` (Kurmandji), `ckb` (Sorani).
- **~1-3 %**. ~1 M Kurdes en Europe, dont une part française significative depuis années 90 (asile politique). Souvent confondus avec Turcs mais L1 ≠ turc.
- **Différencier** : Kurmandji (Turquie/Syrie, dominant en France) >>> Sorani (Irak/Iran, minoritaire).
- Source : Institut Kurde, Solidaires.

### 13. 🇧🇬 Bulgare — autonyme : Български
- ISO : `bg`.
- **~1-2 %**. Détachement croissant post-2007. Souvent intérim BTP second-œuvre.

### 14. 🇺🇦 Ukrainien — autonyme : Українська
- ISO : `uk`.
- **~1-3 %** (en très forte hausse depuis 2022). Protection temporaire UE prolongée jusqu'à mars 2026+. Insertion BTP rapide (réfugiés majoritairement femmes, mais hommes valides → chantiers).
- Régions : IDF, Grand-Est, Pays-de-la-Loire.

### 15. 🇪🇸 Espagnol — autonyme : Español
- ISO : `es`.
- **~1-2 %**. Détachement (n°5) + récente vague économique 2012-2018. Bcp second-œuvre carrelage, façade.

## Notes ethnographiques pratiques

### Dialectes maghrébins → adopter darija et NON MSA
Les ouvriers du chantier comprennent mal l'arabe standard moderne (langue de la presse/coran). La **darija** marocaine / **arq** algérienne sont indispensables. En IDF, mixer darija marocaine + algérienne fonctionne (mutuellement intelligibles à 80%). Tunisien (aeb) plus rare BTP.

### Langues berbères = couche obligatoire à part
~1/3 des "Algériens" en France sont kabylophones L1, ~25% des "Marocains" sont chleuhs/rifains L1. Une plateforme darija-only **rate ces 6-9% de la main d'œuvre**. P1 minimum.

### Subsahariens : bambara + soninké font 70% du besoin
Les foyers IDF parlent bambara comme **lingua franca de chantier** entre Maliens/Sénégalais/Guinéens/Ivoiriens. Soninké en langue communautaire forte. Wolof et peul complètent.

### Lusophones : pt-PT uniquement, JAMAIS pt-BR par défaut
Le portugais européen domine totalement le BTP. pt-BR (brésilien) marginal — utile éventuellement IDF récent (post-2018), mais à proposer en option, pas par défaut.

### Turcs vs Kurdes : 2 communautés distinctes
Ne pas mettre tous les "originaires de Turquie" sous `tr`. Les Kurdes refusent souvent le turc et préfèrent kurmandji. À demander à l'inscription.

### Anglais ?
Marginal en BTP français (sauf détachements UK rares post-Brexit). Ne pas en faire un P0. Utile en P2 pour techniciens spécialisés (chantiers stratégiques).

### Support langues rares (test croisé arxiv 2305.06530 + machinetranslation.com)

- **Bambara / Soninké / Peul / Wolof** : aucun LLM chinois ne les supporte vraiment bien en zéro-shot. Qwen3.6-Max et DeepSeek V4 (avec prompt few-shot et glossaire BTP) donnent les meilleurs résultats, mais qualité « passable » (50-65% BLEU vs human reference). **Recommandation** : pipeline hybride avec glossaire métier BTP + post-édition humaine sur ces 4 langues, pas de traduction LLM pure.
- **Darija (ary/arq)** : DeepSeek a des limites prouvées sur dialectes arabes vs MSA. **Qwen-Max et GLM-5.1 plus à l'aise** sur arabe dialectal. Workaround universel : prompt "translate to colloquial spoken Maghrebi Arabic as used by construction workers in France, NOT modern standard Arabic".
- **Kurmandji** : très médiocre partout. Préférer NLLB-200 (Meta open-source) ou MADLAD-400 en backend dédié.
- **Polonais / Roumain / Bulgare / Ukrainien / Portugais européen / Espagnol / Turc** : tous excellents partout (LLM chinois ≥ GPT-4o sur ces 7 langues).

### Recommandation finale par cas d'usage DEP

1. **Default cheap (95% du trafic, langues européennes + arabe MSA/darija basique)** → `qwen-turbo` via DashScope International. $0.033/$0.10, 1-2s, suffisant pour messages chantier courts (<200 tokens).
   - Fallback : `glm-4.7-flash` (gratuit) en backup si quota Alibaba.

2. **Precision tier (darija dialectale, berbère, africaines)** → `qwen3.6-max` ou `deepseek-v4-pro` avec **prompt engineering chantier** (glossaire métier injecté en cache prompt → -90% coût cache hit).
   - Stable base prompt cache-friendly = essentiel pour rentabilité.

3. **Long-contexte (PDF devis, plans, fiches sécurité multi-pages)** → `deepseek-v4-flash` (1M ctx, $0.14/$0.28). Imbattable rapport prix/contexte.

4. **Langues rares (bambara, soninké, peul, wolof, kurmandji)** → pipeline dédié hors LLM chinois :
   - **NLLB-200 distilled** (Meta, gratuit, self-hosted) en première passe
   - Post-édition `qwen3.6-max` pour fluidifier
   - Glossaire métier BTP injecté
   - **Toujours flagger** "traduction approximative — relisez avec un collègue bilingue" dans l'UI DEP.

5. **Vocal (futur)** → ASR : Qwen3-ASR + SenseVoice (couvre arabe, polonais, roumain, portugais, turc). TTS : MiniMax T2A multilingue, Doubao TTS. Pour langues africaines : externaliser vers Meta MMS / AfriSpeech-200.
